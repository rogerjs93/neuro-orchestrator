Neuro-Orchestrator

One-command neuroimaging pipelines for hypothesis testing.

Research use only. This wraps peer-reviewed, research-grade tools. It is not a medical device and not for diagnosis.

Wraps peer-reviewed tools (fMRIPrep, FastSurfer, MRtrix3, Nilearn) behind a FastAPI web UI and Docker, adding group statistics, a review-gated masking stage, and parametric STL export.

Key features

  • BIDS ingestion plus DICOM to BIDS conversion
  • Per-subject functional connectivity matrices and graph metrics
  • Two-group statistics: permutation FWE, NBS, and FDR, with covariates
  • Review-gated masking with a manual mask editor
  • Parametric STL export with topology repair and uniform remesh

Run with Docker

# Docker Desktop must be running
docker-compose up -d --build orchestrator
# then open http://localhost:8080
# tip: set MOCK_MODE=1 for a light run without pulling large tool images
